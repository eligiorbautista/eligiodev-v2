## About The Project

This project is a complete personal portfolio website built from the ground up. It serves as a central hub for my professional work, including featured projects, career history, certifications, and client testimonials. The design is inspired by retro-futuristic aesthetics, combining modern web technologies with a nostalgic, terminal-like feel.

### Key Features

-   **Single Page Application (SPA)**: Smooth navigation between sections and dedicated pages for projects, certificates, and testimonials.
-   **Fully Responsive**: Optimized for a seamless experience on desktop, tablet, and mobile devices.
-   **Interactive & Animated UI**: Features a custom cursor, subtle hover effects, and on-scroll animations for an engaging user experience.
-   **Dynamic Content Pages**: Separate, filterable, and searchable pages for Projects, Certificates, and Testimonials.
-   **Secure Resume Access**: Password-protected modal for downloading the resume.
-   **Modern Aesthetics**: A unique retro-futuristic theme with carefully chosen fonts, colors, and animations.

## Getting Started

Follow these steps to run the project on your local machine for development and testing purposes.

### Prerequisites

-   [Node.js](https://nodejs.org/) (v18 or later recommended)
-   [npm](https://www.npmjs.com/) (usually comes with Node.js)

### Installation & Setup

1.  Clone the repository to your local machine:
    ```sh
    git clone https://github.com/eligiorbautista/eligiodev-v2.git
    ```
2.  Navigate to the project directory:
    ```sh
    cd your-repo-name
    ```
3.  **Perform a clean installation** of the dependencies.

    *For Windows (PowerShell):*
    ```powershell
    Remove-Item -Recurse -Force .\node_modules; Remove-Item -Force .\package-lock.json; npm install
    ```
    *For macOS/Linux (Bash/Zsh):*
    ```sh
    rm -rf node_modules package-lock.json && npm install
    ```

### Running the App

Once the installation is complete, you can start the development server:

```bash
npm run dev
```

This will start the application, and you can view it in your browser at `http://localhost:3000`. The server supports hot-reloading, so any changes you make to the code will be reflected in the browser instantly.
